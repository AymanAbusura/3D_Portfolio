## 3D Portfolio
This is my 3D portfolio built using React and Vite, showcasing various interactive 3D elements. It leverages modern web technologies like Three.js and React Three Fiber to create a visually engaging experience.

## Tech Stack
* React.js
* Three.js
* React Three Fiber
* React Three Drei
* Email JS
* Vite
* Tailwind CSS
* i18next
* Particles.js

## Features
* Interactive 3D animations and models.
* Responsive design using Tailwind CSS.
* Multi-language support via i18next.
* Contact form integrated with EmailJS.
